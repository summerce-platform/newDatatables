기본 또는 요약 데이터를 원하는지 여부에 관계없이 테이블의 기반이 될 워크 시트를 선택하고 마지막으로 반환 할 레코드 수를 선택합니다. 필수 필드입니다.

참고 : 현재 버전의 Extensions API에 따라 총 레코드 수는 기본 데이터에만 적용됩니다.

이러한 선택을 마치면 다음 탭에서 가능한 열 목록을 볼 수 있습니다. 여기서 열을 재정렬하고 열 제목을 사용자 지정할 수 있습니다.

참고 : 워크 시트를 업데이트하거나 데이터 항목을 추가 또는 제거하는 경우 확장 프로그램을 다시 구성하려면 다시 돌아와야합니다.

재정렬 할 열을 끌어서 놓습니다. 열 이름을 변경하려면 열 중간을 클릭하고 Tableau 데이터 원본 이름을 유지하려면 비워 둡니다.


화면의 정보 밀도를 높이려면 공백의 양을 줄이려면 컴팩트 레이아웃을 선택하세요

Hover에서 행 강조 표시를 선택하여 마우스를 그 위로 가져갈 때 행을 강조 표시하도록 DataTables의 스타일을 지정합니다.

텍스트 줄 바꿈을 선택하면 DataTables의 스타일이 셀에서 텍스트를 줄 바꿈하도록 변경됩니다

정렬에 사용될 때 DataTables의 스타일이 열을 강조 표시하도록하려면 정렬 열 강조 표시를 선택합니다.

열 강조 표시를 선택하여 행 사이에 선을 그리도록 DataTable의 스타일을 지정합니다.

행 스트라이핑 표시를 선택하여 DataTable의 스타일을 행 색상을 대체합니다.

고유 한 테이블 이름 포함을 선택하여 모든 컨트롤의 프로그래밍 방식 이름 내에 테이블 이름을 포함하도록 DataTables에 지시합니다. Tableau Dashboard 내에서 둘 이상의 DataTables Extension이 사용되고 있는지 확인해야합니다.
